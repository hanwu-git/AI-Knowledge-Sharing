## 简介

数据分析是现代办公的核心技能之一。本章节将教你如何利用 WorkBuddy 自动化处理 Excel 数据、生成专业图表，并从中提取有价值的业务洞察。

## 核心能力

WorkBuddy 在数据分析方面支持：

- **数据清洗**：自动处理缺失值、重复数据、格式统一
- **数据统计**：快速计算汇总指标、趋势分析
- **图表生成**：自动生成柱状图、折线图、饼图等可视化图表
- **报告输出**：将分析结果整理成专业的数据报告
- **批量处理**：同时处理多个数据文件

## 实战场景

### 场景 1：销售数据月度分析

**数据**：包含日期、产品、销售额、区域等字段的 Excel 表。

**AI 处理**：
1. 读取 Excel 文件
2. 按月份汇总销售额
3. 计算同比增长率
4. 生成趋势图和区域对比图
5. 输出分析结论

**示例指令**：
```
请分析这份销售数据：
1. 计算每月总销售额和环比增长率
2. 找出销售额 Top 5 的产品
3. 按区域统计销售占比
4. 生成可视化图表（趋势图+饼图）
5. 总结关键发现和业务建议
```

### 场景 2：员工业绩自动排名

**数据**：包含员工业绩指标的多张表格。

**AI 处理**：
1. 合并多张数据表
2. 计算综合得分
3. 自动排序生成排名
4. 生成绩效分布图
5. 输出个人和团队的业绩报告

### 场景 3：客户满意度调查分析

**数据**：问卷调查结果（评分+开放性回答）。

**AI 处理**：
1. 统计各维度满意度得分
2. 分析开放性回答的情感倾向
3. 找出高频关键词
4. 生成可视化报告

## 操作步骤详解

### 步骤 1：准备数据

确保数据格式规范：
- 使用标准表头（第一行是字段名）
- 统一数据格式（日期、金额等）
- 去除明显异常值（可选）

### 步骤 2：上传数据

在 WorkBuddy 中：
1. 点击文件上传按钮
2. 选择 Excel/CSV 文件
3. 等待上传完成

### 步骤 3：下达分析指令

使用结构化的方式描述分析需求：

```
【分析目标】：季度销售数据分析
【数据文件】：已上传的 sales_q2.xlsx
【分析要求】：
1. 按产品线汇总销售额
2. 计算各区域销售占比
3. 找出增长最快的 3 个产品
4. 生成柱状图和饼图
【输出格式】：Markdown 报告 + 图表图片
```

### 步骤 4：迭代优化

根据初步结果，进一步要求：

```
请在图表中添加去年同期对比数据
请计算每个产品的平均客单价
请将分析结果导出为 PPT 格式
```

## 支持的图表类型

| 图表类型 | 适用场景 | 示例 |
|---------|---------|------|
| 柱状图 | 类别数据对比 | 各月销售额对比 |
| 折线图 | 趋势分析 | 销售趋势变化 |
| 饼图 | 占比分析 | 市场份额分布 |
| 散点图 | 相关性分析 | 价格与销量关系 |
| 热力图 | 密度分布 | 区域销售热度 |
| 漏斗图 | 转化分析 | 销售漏斗 |

## 进阶技巧

### 技巧 1：自动化报表

定期生成固定格式的报表：
1. 准备数据模板
2. 保存分析指令
3. 每月只需更新数据，一键生成报表

### 技巧 2：数据预测

基于历史数据进行趋势预测：
```
请根据过去 12 个月的销售数据，
预测下个月的销售额，
并给出预测依据和置信区间。
```

### 技巧 3：异常检测

自动识别数据中的异常点：
```
请分析这份数据，
找出销售额异常偏低/偏高的月份，
并分析可能的原因。
```

## 最佳实践

1. **数据预处理**：上传前检查数据质量，减少 AI 处理时间
2. **分步分析**：复杂分析拆分成多个步骤，逐步验证
3. **可视化优先**：尽量要求生成图表，直观展示分析结果
4. **保存模板**：将常用的分析指令保存为模板，提高效率
5. **结果验证**：AI 计算结果需人工抽查验证，确保准确性
